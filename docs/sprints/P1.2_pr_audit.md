## Points d'attention identifiés (hors périmètre P1.2)

### ⚠️ SMTP built-in Supabase
Warning visible dans le dashboard Supabase :
> "You're using the built-in email service. This service has rate limits and is not meant to be used for production apps."

BraveHeart a Resend ouvert dans un onglet → présomption que Resend est l'option choisie, mais **non configuré** côté Supabase SMTP Settings.

Risque : à mesure que les invitations s'accumulent (lycée Blaïse Cendrars = potentiellement plusieurs dizaines d'invitations en quelques minutes lors d'un import), le rate limit built-in (~4 emails / heure) bloquera les envois.

**Recommandation :** sprint dédié SMTP Resend avant P1.1 (Import CSV élèves). Pas urgent maintenant (un seul DDFPT invité), mais bloquant avant la phase pilote réelle.

### ⚠️ MFA non activé
Le trigger `on_auth_user_created` est actif mais aucun rôle n'a MFA TOTP forcé pour le moment. Question à reposer en P0.9 (Empty states + RoleGuard) ou en sprint sécurité dédié.
