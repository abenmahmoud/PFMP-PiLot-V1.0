## Migration creee

Fichier :

```text
supabase/migrations/20260508174348_fix_current_establishment_id_recursion.sql
```

Contenu fonctionnel :

```sql
CREATE OR REPLACE FUNCTION public.current_establishment_id()
RETURNS uuid
LANGUAGE sql
STABLE
SECURITY DEFINER
SET search_path TO 'public'
AS $function$
  select coalesce(
    case
      when current_role_app() = 'superadmin' then
        nullif(
          (auth.jwt() -> 'user_metadata' ->> 'active_establishment_id'),
          ''
        )::uuid
      else null
    end,
    (select establishment_id from profiles where id = auth.uid())
  );
$function$;
```

Cette migration ne modifie aucune policy, aucune migration historique et
aucune fonction helper autre que `current_establishment_id()`.
