## 4. Étape 4 (audit unprotected_tables) — SQL simple

Cette requête est simple, peut tourner dans le SQL Editor Supabase
directement (pas besoin de transaction) :

```sql
select count(*) as unprotected_tables
  from rls_audit
 where rls_enabled = false or policy_count = 0;
```

Doit retourner **0**.
