## Salut Codex

Le réseau entreprises est un **module stratégique** d'après VISION.md
(MODULE 7) :

> Objectif : Créer une mémoire professionnelle de l'établissement.

C'est un différenciateur fort vs solutions classiques. On le branche
en lecture seule pour ce sprint. La création/édition entreprise viendra
en P1.

---

## Objectif fonctionnel

1. `/companies` (liste) lit Supabase, filtré par tenant via RLS
2. Page détail entreprise (probablement `/companies/$id` à créer si pas
   existant) avec :
   - infos générales (SIRET, adresse, contact)
   - tuteurs liés
   - historique placements (élèves accueillis)
   - statut + fiabilité
3. Recherche par ville, secteur, famille de métiers, fiabilité
4. Filtres : "à relancer", "partenaires forts", "à éviter"

---

## Contraintes (rappel)

- Mode démo intact
- RLS naturelle
- Pas de N+1 sur la liste (ne charge pas tous les tuteurs/placements
  pour chaque entreprise listée)
- typecheck + build verts
